# blogprototype
## Çalıştırmak için adımlar:
### 1. Virtual environment oluştur ve aktifleştir:
python -m venv venv
#### 1.1 Windows için (cmd):
.\venv\Scripts\activate.bat
#### 1.2 Linux için (bash):
source venv/bin/activate
### 2. Gereksinimleri indir:
pip install -r requirements.txt
### 3.   Main.py'yi çalıştır
python main.py
